#### Spec
<p>We are spec!</p>
<strong>code with no errors</strong>
```javascript
// Hello World
```
<strong>code with expected return value</strong>
```javascript
return true;
```
<strong>errors can be returned</strong>
```javascript
return Error('wrong');
```
<strong>errors can be thrown</strong>
```javascript
throw Error('wrong');
```
<strong>expectedValue can be asynchronous</strong>
```javascript
//callback(42);
callback(42);
```
